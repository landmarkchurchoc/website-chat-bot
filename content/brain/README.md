# Organization Brain

Drop Markdown files in this folder (or subfolders) and they become part of the
AI's knowledge base the next time `npm run ingest` runs (which happens on every
deploy).

Guidelines:

- One topic per file, with a `# Title` heading on the first line.
- Write in the church's own voice — the AI treats these files as Landmark's
  authoritative position.
- Good candidates: statement of faith details, FAQs, ministry descriptions,
  baptism/membership/communion practice, positions on common questions,
  staff-written answers to questions people actually ask.
- Content in the shared Google Drive folder should be exported/pasted here as
  Markdown for now (automatic Drive sync is a planned phase-2 feature).
